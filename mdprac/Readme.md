# 나만의 맛집 기록하기🍜

## 프로젝트 소개
이 레포지토리는 내가 직접 방문한 맛집들을 기록하고 관리하기 위한 프로젝트입니다.  
맛집의 위치, 메뉴, 방문 후기 등을 정리하여 나만의 맛집 데이터베이스를 만드는 것을 목표로 합니다.
## 데이터베이스 설계
